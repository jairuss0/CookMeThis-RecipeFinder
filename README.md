# CookMeThis:  A recipe finder website

### Features
1. Users can search different kinds of ingredients along with filter tags and get the recipe results
2. The website provides alternative instructions (vlog source, youtube souce) for the recipes of the meal
3. Users can save their favourite foods using localStorage (client's device)
